추천시스템

-콘텐츠기반 필터링
사용자가 좋아했던 것의 공통점
-협업 필터링
사용자와 비슷한 사람들(그룹)이 좋아하는 것
-하이브리드필터링
위 두개 짬뽕

1. 콘텐츠 기반 필터링
1-1. 아이템 프로파일 구성
아이템을 설명하는 특성들 ( 목적에 따라 달라짐 )
유튜브 = [ 영상주제, 조회수, 좋아요, 인기프레임 .. ]  << 데이터 형식에 구애받지 않음(이종 데이터 가능)

1-2. 특성당 정보 추출
모든 아이템에게 동일하게 계산이 가능하도록 Enbedding해야함 ( 텍스트>NLP / 이미지>image처리 / 숫자>범주,그대로.. )

1-3. 아이템들의 프로파일 생성
사용자의 프로파일( A에 대한 emb, B에 대한 emb .. ) / 아이템 프로파일 ( A에 대한 emb, B에 대한 emb C에 대한 emb .. )
따로 생성

1-4. 유사도 계산 및 추천
사용자가 평가한 것들을 기준으로 "사용자의 emb"를 구함 ( 평균, 가중합 .. ) >> 아이템 프로파일에 있는 것들과 유사도를 확인

2. 협업필터링
사용자들 사이의 '상호작용 데이터'
데이터가 충분해야 성능이 보장이 된다
