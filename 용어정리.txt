편차 : 관측값 - 평균 
관찰값들이 얼마나 흩어져 있는지를 살피기 위해 사용
합이 항상 0 > 제곱을 하자

분산 : (편차들의 제곱)의 평균
편차의 부호를 제거 하기 위함

표준편차 : 분산의 제곱근
분산의 크기를 줄이기 위함 == 크기 원상복귀
어떤 수의 크고 작음을 직관적으로 이야기해줄 수 있다.

모 표준편차 : 모집단의 표준편차
전체 데이터 갯수 (N)으로 나눔

표본 표준편차 : 표본의 표준편차
전체 데이터 갯수 -1 (n-1)로 나눔

CLV : 고객생애가치
신규확보, 기존고객유지 전략 = 이탈 징후 발견
신규확보 유지비용 >> 기존고객 유지비용 = 기존고객유지가 중요 > 기존고객가치를 높이자
+측정방법 : 1.가치창출 터치포인트 식별 2.고객기록 통합 3.터치포인트 수익층정 4.1,2,3더하기 
+공식 : 

CAC : 고객 획득비용
신규확보 유지비용 (마케팅, 광고, 스페셜 오퍼)

NPS : 고객 충성도 측정 (순추천고객지수)
-100 ~ +100 점수 ( 추천고객비율 - 비추천고객비율 ) 추천고객(9~10점) 중립고객(7~8점) 비추천고객(0~6점)
정기적으로 시행, 
+TNPS : 고객과 기업이 상호작용해서 조사
+eNPS : 직원과 기업

CSAT : 고객 만족도 측정
점수가 아닌 % ( 1~5번 보기 ) = ( 4, 5점 응답 / 전체 설문조사 답변 수 * 100 ) 그냥 긍정적인 평균 내는것
ACSI (미국고객만족도지표) 를 활용해서 받은 CSAT 수치가 높은지 판별 가능
정기적으로 측정
+장점 : 측정하기 쉽고, 고객부담적고, 분석하기 쉽고, 널리 알려짐
+단점 : 자가보고 의존, 세부사항 제한, 개인경험 국한, 편향적 답변
