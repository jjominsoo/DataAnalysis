Grid Search / Random Search
> *하이퍼파라미터를 최적화하기 위한 방법론
Grid Search			: 하이퍼파라미터의 구간을 정하고 *Manual Search를 다 함
Random Search		: 하이퍼파라미터를 랜덤으로 뽑아서 진행함, 시도할 숫자, 구간, 횟수를 지정








LASSO				: 데이터 선형 회귀 모델로 과적합[오버피팅]을 막기위해 하이퍼파라미터[정규화계수]를 추가한 모델
CV [Cross Validation]		: 교차검증, 한 사건을 다른시각, 여러자료로 검사해서 정확성을 높이는 방법
RMSE [Root Mean Square Deviation]	: 평균 제곱근 편차, 예측값과 실제환경값과의 차이를 다룰 때 흔히 사용하는 측도
Bayesian optimization	: 
Manual Search		: 일일히 하이퍼파라미터를 조절해 가면서 최적을 찾는 과정  
파라미터			: 모델에 적용시켜 샘플에 대한 예측을 함 (=모델파라미터) [ 결정계수, 절편, 가중치, 서포트벡터 ] 
하이퍼파라미터	: 최적의 모델을 찾기 위한 모델 설정 변수 [ 학습률, 에포크수, 가중치 초기화, 손실함수, 미니배치 크기, 은닉층 갯수, 일반화 파라미터, k-NN의 k ]
