SVM 

두 집단 중 가장 가까운 2개의 데이터를 기준으로 데이터 사이의 거리가 짧게 나누는 선을 찾는 것
데이터 사이의 거리 = 마진 / 선 = 마진을 반으로 가르는 직선
+ 하드 마진 : 데이터가 이미 이쁘게 분류되어 있어서
+ ***소프트 마진 : 현실의 데이터 ( 오차를 고려 ) >> 일반화

***소프트마진
슬랙변수(프사이) : 모든 데이터가 SVM 선 +- 1/2(마진) 에 존재하는지 아닌지를 판단하는 슬랙변수(0/~1/1~)을 가짐
>> 슬랙변수의 최소화를 목적

즉 마진은 최대화 마진위반은 최소화

SVM은 기본으로 선형이지만 안되는 경우가 많음
>> 차원을 늘리면 선형으로 처리가 가능 하지만 계산량이 늘어남
>> 커널트릭

커널트릭
고차원의 장점을 취하면서 계산 복잡도를 그대로 가져감
1. 다항커널		: 다항방정식에서 계수를 높임
2. RBF커널(가우시안커널)	: 두 포인트 사이의 거리 ** 제일 자주 쓰임
3. 시그모이드커널		: 이진 분류 최적화(데이터 잘 나눠있을 때)


Decision Tree

엔트로피
불확실성의 척도 (높을수록 정보량이 적다)

Decision을 잘나눠야한다 확실하게 그룹을 나눌수 있는 조건이어야한다 ( 7/7/7 <<<<<<< 21/0/0 ) == 엔트로피가 낮도록 해야한다.

회귀선에서 outlier를 처리하기 위한 회귀선을 처리하기 위해 max_depth를 줄이자 ( max depth가 높을수록 회귀선이 굴곡이많아짐 ) 보통 2~'3' 씀 ( overfit 조심 )

해석이 편함 ( 어떤 특성에서 몇을 기준으로 나눌수 있는지 알 수 있음  )
성능이 괜찮음
스케일링에 둔감함 (굳이 안해도 됨)


재현율 <> 정밀도 

원하는 결과가 중요할 수록 '재현율'을 따져야한다. ( 우리가 원하는 결과를 내놓는다고 의심되는 경우 알려준다 )
그닥 중요하지 않을 수록 '정밀도'를 따져야한다 ( 100%로 적합할 때만 알려준다 )
