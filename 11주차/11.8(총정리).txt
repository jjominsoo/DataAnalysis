지도학습
1. 회귀
선형회귀				: 데이터의 경향선 .. 정규방정식 / 경사하강법을 사용해 비용(MSE)를 줄이는 형식							

2. 분류
선형분류 				: 회귀와 같은 방법 > logit생성 >> 시그모이드 함수적용 > 확률 값 계산 / 경사하강법을 사용해 비용(Log Loss)을 줄임
svm(svc)				: 데이터 분단선 .. 마진 최대화, 슬랙변수 최소화 + 커널트릭을 이용한 고차원 데이터 처리
decision tree			: 엔트로피(지니불순도)가 작아지는 방향으로 나눔
dt  regression			: MSE값이 작아지는 방향으로 나눔
cnn 				: 정보를 나누고(Convolution필터), 주변 정보를 통합해서 상위 개념 구성하고(Pooling), 반복함
rnn				: 신 정보 + 기존 정보(기억) > 업데이트, 반복

비지도
1. 군집화
k-means				: 비슷한 K개의 무리로 나눔(점차 중심점을 업데이트하는 것으로) .. 엘보우방법(적절한 K 찾기) (sse, 실루엣계수)

2. 이상치 탐지
isolation forest			: 고립 분할의 depth가 적은 것이 이상치
